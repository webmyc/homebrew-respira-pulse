## What is Respira Pulse?

Respira Pulse is a beautiful macOS menu bar app that monitors your Vercel deployments in real-time. Built for developers who deploy constantly and need instant feedback without context-switching.

## Installation

```bash
brew tap webmyc/respira-pulse
brew install --cask respira-pulse
```

## Usage

After installation, Respira Pulse will appear as an icon in your macOS menu bar (top right of your screen).

1. Click the menu bar icon to open the dropdown
2. Connect your Vercel account by entering your Personal Access Token
3. Select which projects to monitor
4. Get instant notifications when deployments start, succeed, or fail

### Getting Your Vercel Token

1. Go to [vercel.com/account/tokens](https://vercel.com/account/tokens)
2. Click "Create Token" and give it a name (e.g., "Respira Pulse")
3. Copy the token and paste it in Respira Pulse settings

Your token is stored securely in macOS Keychain and never shared.
